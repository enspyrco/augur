## The pick

`weave` — the augur verb that finds **warm intro paths in the PUBLIC collaboration graph**, not the private social graph. A warm path is a shared public *artifact* (a co-committed repo, a shared dependency, a co-authored paper, a co-attended event), not a private mutual friend.

## Why it glows (aliveness)

- The co-commit vein is the **exact inverse query** of the existing `github` excavator (`src/excavators/github.mjs`): that one is `handle → artifacts touched`; weave's is `artifact → who else touched it`. Same GitHub API, run backwards. The muscle already exists.
- Structural, not bolt-on: weave-via-artifacts is `recombine`/`compose`'s bisociation engine **pointed one level down** — "what have A and B ALREADY both touched" instead of "what COULD A and B build together." It plays to augur's single strongest capability (forensic public excavation) instead of the private-graph game augur was never going to win.
- Three memory files independently reach for it (`concept_artifact_weave`, `concept_augur_engine`, `project_augur_product_architecture`), and it's named in two open tasks (#2, #4). Not a whim — a convergent pull.

## What it changes (impact)

- **Both ancestor `compose simonw` runs cold-opened.** Weave upgrades `compose(target, us)` from "here's a rhyme in your work" to "we both committed to X — here's the rhyme." Cold email → warm email, and receipted (repo URL), so warm-without-creepy.
- Removes augur's structurally thinnest stage from the "maybe don't build it" pile and makes it arguably the strongest.

## The spark

> The warm-path problem that made `weave` augur's thinnest, most-LinkedIn-dependent stage dissolves the moment you stop asking "who do we both know" and start asking "what have we both touched."

## The falsifier (what would prove this ore is slag)

> If real target×us public-artifact overlap is **empirically sparse** — for most realistic (target, us) pairs the co-commit / shared-dependency / co-authorship intersection is empty — then artifact-weave-into-compose is a garnish that fires on rare pairs, and the honest steady state is compose's cold-open. **Heat MUST measure this**: run the intersection on 3-5 real pairs and count non-empty overlaps. If usually empty, the reframe is elegant but inert on the common case, and *broker-weave* (path-finding to a shared third party) is the real product, not artifact-weave.

This is the load-bearing assumption the enthusiasm rests on. Temper must not let it launder past.
